Structure Tag

TAG Description
<html> ใช้ในการกำหนดว่าเป็นเอกสาร HTML
<head> ใช้ในการใส่ข้อมูลเพิ่มเติม เช่น meta tags, title, link สำหรับ CSS, script สำหรับ JavaScript
<body> ส่วนที่จะแสดงผลเนื้อหาในหน้าเว็บ
<header> ส่วนหัวของเว็บไซต์ เช่น โลโก้ เมนูนำทาง
<footer> ส่วนท้ายของเว็บไซต์ เช่น ลิขสิทธิ์ ข้อมูลติดต่อ
<section> ใช้ในการแบ่งส่วนของเนื้อหาหลักในหน้าเว็บ
<article> ใช้สำหรับบทความหรือเนื้อหาที่เป็นหน่วยที่สามารถแยกออกมาได้
<div> ใช้ในการจัดกลุ่มเนื้อหา เป็นกล่องโครงสร้างแบบบล็อก
<span> ใช้ในการจัดกลุ่มเนื้อหาแบบอินไลน์

Form and Input Tags

TAG Description Example
<form> ใช้ในการสร้างฟอร์มสำหรับรับข้อมูลจากผู้ใช้ <form></form>
<input type="text"> ช่องกรอกข้อความแบบปกติ
<input type="password"> ช่องกรอกรหัสผ่าน (จะซ่อนตัวอักษร)
<input type="email"> ช่องกรอกอีเมล (ตรวจสอบรูปแบบอีเมลโดยอัตโนมัติ)
<input type="number"> ช่องกรอกตัวเลข
<input type="radio"> ปุ่มเลือกแบบตัวเลือกเดียว (radio button)
<input type="checkbox"> ช่องเลือกหลายตัวเลือก (checkbox)
<input type="submit"> ปุ่มส่งข้อมูลฟอร์ม
<textarea> ใช้สำหรับช่องกรอกข้อความหลายบรรทัด
<select> และ <option> ใช้สำหรับสร้างรายการเลือก (dropdown)
<button> ปุ่มที่ใช้สำหรับทำกิจกรรมต่าง ๆ (เช่นคลิกเพื่อส่งฟอร์ม)
