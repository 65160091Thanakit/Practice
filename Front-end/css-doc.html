Property Description Example

Inline
position กำหนดตำแหน่งของ element เช่น relative, absolute, fixed
Relative

Property Description Example
width กำหนดความกว้างของ element
Width 100px
height กำหนดความสูงของ element

Margin 20px

พื้นที่ด้านนอกของกรอบ element (เส้นประฟ้า) คือ margin
